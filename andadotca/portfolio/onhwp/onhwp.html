Ontario New Home Warranty Program "ONHWP"

"Order Express" On-line Shopping

  • [Consumer]: (archived: white paper) Client views products and places items in shopping basket. Items can be added or removed from shopping basket at any time. Client checks out by "placing order". Information about the client is entered in a form and validated before it gets sent to the server for processing. The client receives an order number from the server, along with payment methods information page.
  • [Company Administration]: These examples just show the layout of the forms that a company administrator uses. The active programs on the server are removed. The administrator can use an internet interface to change the way the opening page looks, change the product catalogue and process customer information that has been received over the internet.
  • [Animation & Graphics] ONHWP dog icon. Animation sequences and buttons.
Technologies used: SQL database (originally Access), ASP, VBScript, JavaScript, Shockwave Flash, Corel graphics
