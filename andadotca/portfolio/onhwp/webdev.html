WEB DEVELOPMENT

In general, I like to describe a web site by what it does (Does it have active, dynamic or static content). This gives a more intuitive platform to decide what you would like on your web site, as well as a better feel for what it would cost. I also have detailed information of technologies I use.

Active Content
User information is processed and stored - in database, file, spreadsheet, etc. - and this decides what is posted next.
Dynamic Content
User interaction causes things to change - like shape, color, movement - but this information is not processed for future use.
Static Content
Web page text, images and standard anchors/links.
