Welcome to "Self-Directed Coaching" platform, a program that applies sports psychology principles to high level sport. This program teaches athletes sound motor learning and performance methods to help in their consistency and in their planning for new world level results.

Self-Directed Coaching Program provides methods to create cognitive cues to sequential muscle movements. Whereas kinesiology provides methods for muscle health, the SDC program provides mental control methods to learn muscular sequences and perform those same sequences on demand. When a body changes through natural growth, training or physiotherapy, it needs to explicitly attach that new physical essence to cues that control larger muscle sequences. It is this last step that the SDC program specializes in.

The program was created in 1996 along with a website explaining the program. A printed Manual was created in 2011, and an online version in 2015. The manual has 4 sections: section 3 is of main interest to elite athletes, while sections 1, 2 and 4 are supportive sections.
